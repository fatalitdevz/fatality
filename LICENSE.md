### ❌ High CPU Usage After Injection

**Cause:** Particle and decal rendering enabled alongside ESP
**Fix:** Enable performance mode inside the menu (`Visuals > Performance > Enable Performance Mode`). Disable decals, particles, and water effects.

### ❌ macOS: Loader Crashes on Launch

**Cause:** Missing Visual C++ Runtime inside Wine container
**Fix:** Install `vcredist_x64_2019.exe` inside the CrossOver container before running the loader.

### ❌ Android: Menu Not Visible

**Cause:** Emulator DPI too low for overlay rendering
**Fix:** Set emulator DPI to 96+ and resolution to at least 1920×1080 in emulator display settings.
